# User Guide

## User Flow

1. Open the dashboard.
2. Authenticate a Freighter wallet.
3. Create a loop with counterparty, role, expiry, and approval policy.
4. Confirm the loop when work starts.
5. Capture participant approvals.
6. Close the loop when approvals are complete.
7. Review events, metrics, and monitoring.

## Key Pages

- Dashboard: wallet status and latest loops
- Loops: all loops and user registry
- Loop detail: approvals and lifecycle actions
- Onboarding: participant capture and CSV export
- Metrics: trust and transaction trends
- Monitoring: service health and security checklist

## Error Cases

- wallet network mismatch
- expired API session
- participant authorization failure
- approval requirement not met for closure
- origin not allowed
