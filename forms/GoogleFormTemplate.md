# Google Form Template

## Purpose

Collect structured product, UX, security, and wallet feedback for TrustLoop.

## Suggested Form Title

`TrustLoop Feedback`

## Suggested Description

Tell us what worked, what failed, and what should improve. Do not submit seed phrases, private keys, recovery codes, or other secrets.

## Questions

1. Role
2. Name
3. Email
4. Feedback type
5. Priority
6. Feature or issue summary
7. Detailed description
8. Steps to reproduce
9. Wallet used
10. Network used
11. Transaction hash or explorer link
12. Suggested improvement
13. Consent confirmation that no secrets were shared

## Export Workflow

Create the form in Google Forms, link responses to Google Sheets, then export response snapshots into `feedback/` as `.xlsx` or `.csv`.
