# TrustLoop User Guide

## Create a Loop

1. Connect Freighter on Stellar Testnet.
2. Open the dashboard.
3. Click `New Loop`.
4. Enter the counterparty wallet address.
5. Choose role and approval policy.
6. Submit the demo transaction.

## Track Loop Health

- Dashboard shows active, pending, and completed loops
- Events page shows trust event history
- Loop detail page shows score, expiry, and audit trail

## Use the Advanced Feature

The advanced feature is a multi-party approval workflow.

Before an active loop is closed:

- client approval can be captured
- freelancer approval can be captured
- close-readiness is shown explicitly

This creates a stronger handoff model for milestone-based collaboration.

## Review Metrics

Open `/metrics` to view:

- active users
- verified wallets
- recent transaction volume
- completion rate
- trust score averages

## Review Monitoring

Open `/monitoring` to view:

- API uptime
- latency
- indexed event counts
- service states
- active alerts
- security checklist

## Manage User Onboarding

Open `/onboarding` to:

- capture participant details
- store wallet addresses and feedback
- export a CSV for reporting or submission evidence
